In OSPF (Open Shortest Path First), an LSA (Link-State Advertisement) is the basic unit of information exchanged between OSPF routers. LSAs are used to describe the state of links (interfaces), networks, and routes in the OSPF domain. Routers exchange LSAs to build the Link-State Database (LSDB), which is used to calculate the shortest path tree using Dijkstraâ€™s SPF algorithm.
